2 XProc 3.0

XProc 3.0 is defined normatively by two specifications:

  • XProc 3.0: An XML Pipeline Language defines the semantics of XProc pipelines.

  • XProc 3.0: Standard Step Library defines the required atomic steps. All conformant implementations implement these steps.

3 Additional Steps

There are many additional, optional XProc steps.

  • XProc 3.0: validation steps

  • XProc 3.0: operating system steps

  • XProc 3.0: file steps

  • XProc 3.0: text steps

  • XProc 3.0: dynamic pipeline execution

  • XProc 3.0: paged media steps
